# Pinguine Pizzaria

Site de uma pizzaria desenvolvido durante estudos para apresentar um projeto simples em HTML e CSS.

## Visão geral

Este projeto é composto por páginas estáticas que simulam um site de pizzaria com navegação entre cardápio, menu de sobremesas, bebidas e uma seção de promoções. O objetivo é demonstrar estruturação de conteúdo, uso de links, imagens e estilos visuais com CSS.

## Páginas principais

- `main.html` - página inicial com destaque para produtos mais vendidos e promoções.
- `menu_pizza.html` - menu de pizzas salgadas.
- `menu_bebida.html` - menu de bebidas.
- `menu_sobremesa.html` - menu de sobremesas.
- `cardapio.html` - visão geral do cardápio.
- `produto.html` - página de detalhes de produto.
- `login.html` - página de login/inscrição.
- `carrinho.html` - página de carrinho de compras.
- `monte_iceberg.html` - página especial de montagem de pedido.

## Estrutura de arquivos

- `styles.css` - estilos gerais do site.
- `images/` - imagens usadas nas páginas, incluindo a logo da pizzaria em `images/logo.jpg`.
- `icons/` - ícones utilizados na navegação e nos botões.
- `LICENSE` - arquivo de licença do projeto.

## Como usar

1. Abra a pasta do projeto.
2. Execute o arquivo `main.html` no navegador.
3. Navegue pelas páginas usando o menu superior.

## Observações

- O projeto é estático e não possui backend nem formulários funcionais.
- A navegação entre páginas usa links diretos para arquivos HTML locais.
- O favicon do navegador está configurado para usar `images/logo.jpg`.

## Como contribuir

- Faça melhorias nos estilos em `styles.css`.
- Atualize o conteúdo e imagens das páginas HTML.
- Verifique se os links entre páginas estão corretos.
